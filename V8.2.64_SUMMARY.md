## 🐛 Bugs Fixed

### 1. **Story - Button not appearing**
**Root cause:** `window` parameter in `didMoveToWindow:` can be nil or not a UIWindow  
**Fix:** Use `[UIApplication sharedApplication].keyWindow` for positioning  
**Code:** `Tweak.x:1166-1210`

### 2. **Newsfeed - Controller not found**
**Root cause:** Hooking wrong class (`VideoContainerView` doesn't exist in 560.x)  
**Fix:** Hook `FBVideoPlaybackContainerView` and check `_videoPlaybackController` ivar  
**Code:** `Tweak.x:1719-1750, 1795-1828`

### 3. **Reels - setPlaying: hooking wrong class**
**Root cause:** Hooking `setPlaying:` on ANY class that responds (including `FBMemModelObjectUnknownSelectorHandler`)  
**Fix:** Filter to only hook `FBVideoPlaybackController`  
**Code:** `Tweak.x:2516-2531`
